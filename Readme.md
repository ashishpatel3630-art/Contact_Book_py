# 📒 Contact Book Management System

A simple **Contact Book Management System** built using **Python** that allows users to store, view, update, and delete contacts. The project uses **JSON file handling** to permanently save contact information.

---

## 🚀 Features

✅ Add new contacts  
✅ View all saved contacts  
✅ Update existing contacts  
✅ Delete contacts  
✅ Store data permanently using JSON file  
✅ Simple command-line interface (CLI)  
✅ Beginner-friendly Python project  

---

## 🛠️ Technologies Used

- Python 3
- JSON
- File Handling
- Functions
- Dictionary Data Structure
